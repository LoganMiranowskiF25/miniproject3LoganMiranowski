# Mini-Project 3

A small flask app to manage user registration, login, and posting of content to a dashboard.

### Description

This project is a flask web app designed to:

- accept user login/registration information
- allow for users to posts their thoughts to a dashboard

The goal was simply to practice flask development

### Getting Started

### Dependencies

* Python 3.10+
* Flask
* Jinja2
* SQLite3
* Bootstrap 5 (from CDN in templates)

> All dependencies listed in: 'requirements.txt'

### Installation
